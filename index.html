Sets de parametros

Como aclaramos anteriormente, cada sorter fue sometido a pruebas con diferentes sets de parametros. Estos parametros incluyen las siguientes verticales: dataType (tipo de dato del set a ordenar, puede ser String o Integer), dataOrder (orden del set de datos antes de ser ordenado, Ascendiente, Descendiente, Random) y un N (tamaño del set de datos). Para cada sorter se realizaron pruebas para todas las combinaciones de las variables anteriormente mencionadas. De esas pruebas obtenemos informacion de las siguientes variables a analizar: Comparisons (cantidad de comparaciones que realizo el sorter), Swaps (cantidad de intercambios de elementos que realizo el sorter) y Time(ms) (tiempo que tardo el sorter en ordenar el set de datos, expresado en milisegundos).

Bubble Sorter

La Ordenación de burbuja (Bubble Sort en inglés) es un sencillo algoritmo de ordenamiento. Funciona revisando cada elemento de la lista que va a ser ordenada con el siguiente, intercambiándolos de posición si están en el orden equivocado.

Parámetros teoricos

  • Casos promedio y peor:
    • Comparaciones: ~N2/2
    • Intercambios: ~N2/2
  • Mejor caso:
    • Comparaciones: ~N2/2
    • Intercambios: 0

Insertion Sorter

El ordenamiento por inserción (insertion sort en inglés) es una manera muy natural de ordenar para un ser humano. Inicialmente se tiene un solo elemento, que obviamente es un conjunto ordenado. Después, cuando hay k elementos ordenados de menor a mayor, se toma el elemento k+1 y se compara con todos los elementos ya ordenados, deteniéndose cuando se encuentra un elemento menor (todos los elementos mayores han sido desplazados una posición a la derecha) o cuando ya no se encuentran elementos (todos los elementos fueron desplazados y este es el más pequeño). En este punto se inserta el elemento k+1 debiendo desplazarse los demás elementos.

Parámetros teóricos

  • Peor caso:
    • Comparaciones: ~N2/2
    • Intercambios: ~N2/2
  • Caso promedio:
    • Comparaciones: ~N2/4
    • Intercambios: N2/4
  • Mejor caso:
    • Comparaciones: N-1
    • Intercambios: 0

Selection Sorter

El ordenamiento por selección (Selection Sort en inglés) es un algoritmo de ordenamiento que busca el mínimo elemento de la lista, lo intercambia por el primero, busca el siguiente mínimo, lo intercambia por el segundo y así en general. Buscar el mínimo elemento entre a posición i y el final de la lista, intercambiar el mínimo con el elemento de la pocisión i.

Parámetros teoricos

  • Comparaciones: ~N2/2
  • Intercambios: ~N-1
